> 最后更新: 2026-03-28

## 项目地址
- n8n 实例: https://n8n.eeba.cn/
- eHunt 选品数据仪表盘: https://n8n.eeba.cn/etsy.html

## 项目概述
这是一个基于 n8n 的 **Etsy 选品 SOP 自动化系统**，实现了从市场调研到产品分析的完整自动化流程。

## 系统架构
```
SOP主调度器
  ├── 步骤1: 自动挑类目
  ├── 步骤2: 关键词挖掘
  ├── 步骤3: 蓝海评分
  ├── 步骤4: 反查产品
  ├── 步骤5-9: 深度分析
  ├── 步骤10-12: AI生成
  └── 步骤13: 日报生成

支撑服务:
  ├── 配额检查与扣减
  ├── 配额每日重置
  ├── eHunt-Token自动刷新
  ├── 竞品动态监控
  ├── AI辩论系统
  ├── 每日选品摘要邮件
  ├── 邮件通知
  ├── 周报月报生成
  └── 全局错误通知
```

## 技术栈
- n8n (工作流自动化)
- eHunt API (Etsy数据分析)
- AI模型 (选品分析、内容生成)
- 邮件通知系统
